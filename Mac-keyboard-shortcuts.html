Mac Function Key Note: On Mac computers, you'll need to press fn + the function key (e.g., fn+F8) unless the keyboard settings have been changed to use function keys without the fn modifier.

Project Setup & Navigation

Shortcut Function
Cmd + N New ReaClassical project
C Mission Control
Cmd + F5 ReaClassical preferences
H Open manual
P Project settings
Shift + U Check for updates
Home Go to project start
End Go to project end
` View whole project (horizontal and vertical)
Cmd + Opt + → View whole project horizontally
Cmd + Opt + ↓ View whole project vertically
Enter Find take by number
Tab Jump to time
Cmd + ↑/↓ Adjust peaks display zoom
Q Previous item edge/crossfade
W Next item edge/crossfade
, Previous marker
. Next marker
M Add generic marker
Cmd + M Add named marker

Recording

Shortcut Function
fn + F9 Classical Take Record (toggle)
Cmd + Space Pause recording
Opt + fn + F9 Increment take number
Cmd + Enter Open Record Panel
B Open Meterbridge
Opt + R Recording indicator

Take Organisation

Shortcut Function

Basic Editing

Shortcut Function
A Audition at mouse position
Shift + A Audition with custom playback rate
D Show child tracks
E Hide child tracks (ensconce)
S Split item at edit cursor
Shift + S Split items into subtakes
Backspace Delete with ripple
Cmd + Backspace Delete leaving silence
