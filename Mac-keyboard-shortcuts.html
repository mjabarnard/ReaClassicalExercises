Mac Function Key Note: On Mac computers, you'll need to press fn + the function key (e.g., fn+F8) unless the keyboard settings have been changed to use function keys without the fn modifier.

Project Setup & Navigation

Shortcut Function
Home Go to project start
End Go to project end
` View whole project horizontally
Cmd + ' View whole project vertically
Return Find take by number
Cmd + ←/→ Adjust peaks display zoom
Q Previous item edge/crossfade
W Next item edge/crossfade
, Previous marker
. Next marker

Recording & Take Organisation

Shortcut Function
F9 Classical take recording mode
Cmd + Space Record-pause
Option + F9 Increment take number (during silence)
Cmd + = Rank take higher
Cmd + - Rank take lower
Cmd + 0 Remove take ranking
Cmd + Return Open take counter
T Prepare takes
K Colourise items

Basic Editing

Shortcut Function
A Audition at mouse position
Option + A Audition with custom playback rate
D Show child tracks
E Hide child tracks
S Split item
J Join items
G Group items
U Ungroup items

Crossfade Editing

Shortcut Function
F Open crossfade editor
X Create classical crossfade
Z Extend crossfaded items (in editor)

Utilities

Shortcut Function
F5 ReaClassical preferences
F6 Show/hide toolbar
H Open manual
O Open preferences
P Project settings
Option + U Check for updates

Source-Destination (S-D) Editing

Shortcut Function
1 Add destination IN marker
2 Add destination OUT marker
3 Add source IN marker
4 Add source OUT marker
5 Make S-D edit
F3 Assembly line edit
F4 Insert with time-stretching
Cmd + Del Delete all S-D markers

Mastering

Shortcut Function
Cmd + M Enter mastering mode
Cmd + I Enter automation mode
I Insert automation
Y Generate DDP markers
C Generate CUE file and reports
R Render

Commands for macOS | Windows users should substitute Ctrl for Cmd and Alt for Option
