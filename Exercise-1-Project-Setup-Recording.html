Introduction

This exercise introduces you to setting up a ReaClassical project with vertical workflow, configuring input routing, track naming, and the take recording system. It's designed to be completed in the Lyons Studio using the installed Decca tree in SJLCH for audio input (we don't need to adjust the tree for this exercise), but feel free to use different mic inputs if available.

Keyboard Shortcuts: This guide shows Mac shortcuts first with Windows equivalents in brackets. On the studio Mac, you'll need to press fn + the function key (e.g., fn+F5) unless the keyboard settings have been changed to use function keys without the fn modifier.
Understanding Vertical Workflow: The vertical workflow arranges takes in vertical folders - source folders for recording takes and a destination folder for your final edit. This arrangement makes it easy to compare different performances and compile parts of each take into a final version without excessive horizontal scrolling. This approach is common in classical music production and has been implemented in ReaClassical for REAPER.

Part 1: Project Setup with Vertical Workflow

Project Structure: When setting up a project in ReaClassical, you're creating a structure designed for classical music recording and editing. The destination folder (top folder) will eventually contain your final edited version. The source folders below it will contain your individual takes, providing the raw material for your edits. All tracks are linked to a set of mixer tracks that control their settings globally. Mission Control (accessible via C) is your central hub for all track and project management tasks.
Submix Tracks (Optional): Submixes are useful when collecting microphone arrays composed of multiple mono inputs. For example, the Decca Tree uses three mono microphones (Left, Centre, Right) that need to be combined into a stereo output. Creating a submix gives you a single fader to control the entire array. For simple stereo pairs or mono sources, submixes are unnecessary.
  1. Start ReaClassical and create a vertical workflow:
    1. Start ReaClassical and immediately save your project (Command+S [Ctrl+S]) with a descriptive name like "DeccaTree_Session1"
    2. Press Command+N [Ctrl+N] to create a new ReaClassical project
      • When prompted, enter "3" for the number of tracks - this will create three microphone channels in each folder
      • Select "Vertical" workflow when asked
    3. Mission Control will open automatically - this is your central control panel for all track management
    4. Name the tracks "Left", "Centre", and "Right" for the Decca Tree (this naming convention helps with organisation)
    5. Since the Decca Tree uses three mono inputs that need combining, click "Add Special Tracks", find "Submix", enter "1" for the number, and name it "Decca Mix" when prompted
      • Note: This step is optional and only needed for multi-mono arrays like the Decca Tree
    6. Close Mission Control to view your project
  2. Examine your project structure:
    1. Notice the destination folder at the top - this is where your final edited version will eventually be built
    2. Below are six source folders, all with the same track structure - these will hold your individual takes
    3. Each folder contains the same channels (Left, Centre, Right) - this mirroring is key to the vertical workflow
      • Additional source folders will be automatically created as needed when recording multiple takes
    4. Look at the mixer panel (lower part of the screen by default) to see the mixer tracks (prefixed with "M:") which control all folder tracks
      • The #Decca Mix track appears in the mixer - this submix will combine all three Decca Tree microphones
    5. Try expanding/collapsing different track folders to see the vertical arrangement and track relationships
  3. Understanding Mission Control:
    1. Press C to open Mission Control
      • This is your central hub for all track and project management in ReaClassical
    2. Notice the different sections and options available:
      • Track list with names, record inputs, and routing controls
      • Buttons to add/remove/reorder tracks
      • Special tracks management
      • Visibility controls for folders and special tracks
      • Quick access buttons to other RC windows
    3. Take a moment to explore the interface - all project management happens here
    4. Close Mission Control by pressing C again or clicking the X
  4. Configure microphone inputs in Mission Control:
    1. Press C to open Mission Control
    2. In the track list, you'll see input assignment controls for each track
    3. Click "Auto Rec Inputs" to automatically assign recording inputs based on track names
      • Check that it assigns inputs 1-3 for the three tracks (these correspond to the physical Decca Tree microphones)
      • If auto-assignment doesn't work as expected, you can manually select inputs using the dropdowns next to each track name
    4. Close Mission Control
    Tip: Mission Control's input assignment is the quickest way to configure recording sources. The "Auto Rec Inputs" feature intelligently matches track names to available inputs, saving time during session setup.
  5. Alternative: Exploring the routing matrix for input visualisation:
    1. Press Option+R [Alt+R] to open the routing matrix
      • Notice how you can see all tracks and their connections in a single view
    2. The routing matrix provides an alternative method for viewing and selecting microphone inputs
      • This is particularly useful when you need to visualise input assignments across many tracks or when working with Dante preamps and other broad input sources
    3. Notice the 'Source' and 'Destination' rows and columns - you can see and modify microphone input assignments by clicking on different channels
      • You can create mono inputs or stereo pairs
    4. While Mission Control is typically more efficient for input assignment, the routing matrix excels at providing a comprehensive overview of your signal flow
    5. Close the routing matrix window when finished by clicking the X or pressing Escape
  6. Add new tracks using Mission Control:
    1. Press C to open Mission Control
    2. Click "Add Track to All Folders"
    3. When prompted, enter the track name "Audience" - this could represent an audience microphone in a live recording
      • Notice how the track appears in every folder and in the mixer
    4. In Mission Control, select a stereo input for this track using the input dropdown (choose from the Stereo/MC section of the input list)
    5. Now try adding another track called "Soloist" using the same method, and select a mono input for it
    6. Observe how both new tracks maintain the same position in every folder - maintaining consistency across your project
    7. Close Mission Control
    Tip: Mission Control ensures that all track additions, deletions, and modifications are applied consistently across your entire project, saving time and reducing the likelihood of errors that could occur when configuring tracks individually.
  7. Rename tracks using Mission Control:
    1. Press C to open Mission Control
    2. In the track list, rename "Left" to "Decca Left" by clicking on the name and editing it
    3. Similarly rename "Centre" to "Decca Centre" and "Right" to "Decca Right"
    4. Also rename "Audience" to "Hall Ambient" and "Soloist" to "Solo Spot"
    5. Close Mission Control
      • Notice how the names in both the mixer panel and all folders update to reflect your new naming scheme
      • Mission Control automatically propagates all changes across the project
  8. Setting up the Decca Tree submix with routing:
    1. Press C to open Mission Control
    2. In the track list, locate the three Decca microphone tracks (M:Decca Left, M:Decca Centre, M:Decca Right)
    3. For each Decca track, click the routing button (appears as a small routing icon next to the track)
      • In the routing menu, select "#Decca Mix" as the destination
      • Repeat for all three Decca tracks
      • This creates a signal flow where all three microphones feed into the single Decca Mix submix
    4. Close Mission Control
    5. Now configure the spatial positioning of these microphones in the mixer panel:
      • Select "M:Decca Left" and adjust its pan control fully left
      • Select "M:Decca Right" and adjust its pan control fully right
      • Leave "M:Decca Centre" centred in the stereo image
      • This creates a proper stereo image that matches the physical layout of the Decca Tree
    6. Try adjusting the volume faders for each microphone to change their relative levels in the mix
    7. The "#Decca Mix" track now contains a properly configured mix of your three Decca Tree microphones
    Tip: Submixes give you a single fader to control grouped microphones, which is convenient for both recording and editing. In larger orchestral sessions, you might create submixes for strings, woodwinds, brass, etc., allowing for balanced section control while maintaining access to individual microphones when needed. Remember: submixes are only needed when combining multiple mono inputs - you don't need submixes for stereo pairs or single mono sources.

Part 2: Recording and Take Management

Take Recording in ReaClassical: The Classical Take Recording system in ReaClassical handles record arming, group switching, and take organisation automatically. This allows you to focus on the music instead of the technical details. Each take is recorded into its own source folder, making organisation intuitive and editing straightforward later on. This approach is commonly used in classical recording sessions where multiple complete takes are standard practice.
  1. Set up for recording with the Record Panel:
    1. Position the edit cursor at the beginning of the timeline by pressing Home
    2. Select the Destination folder if it's not already selected
    3. Open the Record Panel window with Command+Return [Ctrl+Enter]
      • Notice the interface showing take number, session controls, and ranking options
    4. Right-click on the Record Panel window to access additional options
      • Enter a session name like "DeccaSession" - this helps organise files and makes later identification easier
      • Leave the override take number setting at 0 (automatic)
      • Explore the other options in this dialog - note how you can schedule recordings or set custom take numbers
    5. Click OK to confirm your settings
    Record Panel Buttons: The Record Panel provides mouse-clickable buttons for all recording functions: "Rec" to arm/record/stop, "Pause" to pause recording, "+Take" to increment the take number, and "Next" to move to the next folder. These buttons provide an alternative to keyboard shortcuts. You can also add ranking and notes while recording or after the fact by clicking on a previously-recorded item.
  2. Record multiple takes with automatic folder management:
    1. Click the "Rec" button (or press fn+F9) to arm the tracks for recording - notice how the track meters become active
      • Observe that only the tracks in the first source folder are armed
      • The Record Panel displays the current take number in green
    2. Click "Rec" again (or press fn+F9) to begin recording - the Record Panel now shows a red recording indicator
      • Record about 15-20 seconds of audio from the Decca tree in the Hall
    3. Click "Rec" (or press fn+F9) to stop recording
    4. Notice a few things happening automatically:
      • The next source folder is automatically selected and armed - ready for the next take
      • The Record Panel increments to the next take number
      • All your recorded audio is properly organised in the first source folder
    5. Record a second take using "Rec" (or fn+F9) and observe the same automatic processes
    6. For your third take, try clicking "Pause" (or pressing Command+Space [Ctrl+Space]) during recording to pause
      • Notice that pausing keeps the same take but creates a gap in the recording
    7. Click "Pause" again (or press Command+Space [Ctrl+Space]) to resume recording
    8. Finally click "Rec" (or press fn+F9) to stop recording
  3. Additional recording techniques:
    1. Click "Rec" (or press fn+F9) to begin recording take 4
    2. After a few seconds of recording from the Decca tree, click "+Take" (or press Option+fn+F9 [Alt+F9])
      • This increments the take number without stopping recording - useful for continuous recording sessions
      • Notice how the take counter advances but recording continues
    3. Record a few more seconds and click "Rec" (or press fn+F9) to stop
      • Notice how ReaClassical has now created multiple takes across different source folders
      • If you look at the source folders, you'll see each contains the appropriate audio for each take
    4. Close the Record Panel when you've finished recording
    Tip: The "+Take" button (or Option+fn+F9 [Alt+F9]) is particularly useful during sessions with repeated attempts at difficult passages. It allows the recording to continue while still organising takes properly for later editing.
  4. Auditioning recorded material:
    1. Navigate to take 1 by pressing Return and entering "1"
    2. Hover your mouse over the "Decca Centre" track in take 1 and press A to hear just that microphone
    3. Now hover over the "Decca Right" track and press A to hear just that microphone
    4. Now hover over the parent folder track and press A to hear the complete mix of that take
    5. Try hovering at different positions within the waveform and pressing A - playback starts from that position
    6. Press Shift+A to audition at a slower playback rate (as set in preferences)
    7. Use Space to stop playback at any time
    8. Navigate to take 2 using Return and "2", then repeat the auditioning process
      • Compare how take 1 and take 2 sound by quickly switching between them
      • Notice how auditioning automatically disables any record-armed tracks for safety
    Tip: The auditioning system allows for quick comparison between takes and microphone positions without changing track selection or using solo/mute buttons. This helps you make informed decisions by making comparisons immediate and intuitive.
  5. Ranking takes for editing efficiency:
    1. After auditioning all your takes, navigate back to take 1 by pressing Return and entering "1"
    2. In the Record Panel, click on the take 1 item in the list to select it
      • Notice the ranking dropdown menu becomes available
    3. Click the ranking dropdown and select a rating such as "Very Good"
      • Notice the colour change on the item in both the Record Panel and on the timeline
      • Available rankings are: Excellent, Very Good, Good, Okay, Below Average, Poor, Unusable, False Start, and No Rank
    4. Click on take 2 in the Record Panel list and select "Excellent" from the dropdown
      • Notice how different rankings produce different colours
    5. Click on take 3 and select "Below Average" from the dropdown
      • Note the different colour indicating a lower-ranked take
    6. Click on take 1 again and select "No Rank" to remove the ranking - returning it to the default colour
    7. Explore the different ranking options by applying them to your various takes
      • Notice how the visual colour coding creates an overview of take quality across your project
    8. You can also add text notes to takes by clicking in the notes field in the Record Panel and typing your observations
    Tip: Take ranking is useful during longer recording sessions and later editing stages. The visual colour coding helps identify your best and worst takes quickly. Rank takes as you record them to preserve your initial impressions when the performances are fresh in your mind. Notes can capture specific details like "good dynamics bar 12" or "tuning issue at rehearsal mark C".

Part 3: Exploring ReaClassical's Additional Features

ReaClassical's Navigation and Workflow Features: ReaClassical includes tools for efficient navigation and project management. The ability to quickly locate takes by number, audition individual tracks or complete mixes with a single keypress, and switch between different view modes all contribute to a workflow optimised for classical music production. Understanding these additional features will help you work more efficiently with classical music recordings.
  1. Navigation and visibility controls:
    1. Press ` (backtick) to see the entire project horizontally - all your takes are visible at once
    2. Select any parent folder track and press E to hide all child tracks (ensconce)
      • Observe how the view simplifies to show only the parent tracks - useful for getting an overview
    3. Press D to display all child tracks again - returning to detailed view
      • Try hiding children on some folders and showing them on others to create custom views
    4. Press TAB to open the jump to time dialog
      • You can enter an absolute time or a take number to quickly navigate
      • Try entering different take numbers to quickly jump between takes
      • You can also enter times with decimals (e.g., "2.5") to position partway through
    5. Use Q and W to navigate to the previous or next item edge - useful for moving between edits
    6. Experiment with Ctrl+Up Arrow and Ctrl+Down Arrow to adjust waveform zoom - see how it affects visibility without changing audio volume
  2. Working with track groups and understanding folder structures:
    1. Press Command+D [Ctrl+D] to duplicate a folder without items - creating a new empty source group
      • Notice how the new folder maintains the same track structure as the others
      • This is useful when you need more source groups for additional takes beyond what was automatically created
    2. Open Mission Control with C
    3. Click "Add Special Tracks" to access special track options
      • Add a "RoomTone" track - useful for maintaining natural ambience between edits
      • Add a "Reference" track - helpful for comparing your work to other recordings
      • Notice how these special tracks appear differently in the mixer and serve specific functions
    4. Close Mission Control
    5. Import a short audio file to the reference track and audition it to understand its purpose
  3. Recording features and scheduling:
    1. Open the Record Panel with Command+Return [Ctrl+Enter]
    2. Right-click on the Record Panel and explore the scheduling options
    3. Set a start time 2 minutes ahead of the current time
      • Observe how the Record Panel now displays information indicating when recording will begin
      • This feature is useful for timed recordings or when you need to set up before a performance begins
      • You can also set an end time or duration for automated recording sessions
    4. Cancel the scheduled recording by right-clicking the Record Panel and clearing the time fields
    5. Close the Record Panel
    6. Access the ReaClassical manual with H for detailed information on all features
  4. Exploring the Meterbridge:
    1. Press B to open the Meterbridge
      • Notice the numeric peak holds displayed for each channel
      • When tracks are record-armed, the main window displays recording inputs
    2. Right-click in the Meterbridge window to access options
      • Try checking "Graphical Meters" to see a different display
      • Explore color threshold settings
    3. Close the Meterbridge when finished

Summary

This exercise has introduced the fundamentals of ReaClassical project setup and recording workflow. You have learned to:

  • Create a vertical workflow project with multiple tracks
  • Use Mission Control (C) for track management, naming, and input routing
  • Configure optional submixes for multi-mono microphone arrays
  • Record multiple takes using the Classical Take Recording system (fn+F9)
  • Navigate between takes and audition recordings (A key)
  • Rank takes for editing efficiency

Key Points

  • Mission Control (C) is the central hub for all project management
  • Submixes are only needed when combining multiple mono inputs
  • fn+F9 controls the entire record workflow (arm → record → stop)
  • The Record Panel tracks take numbers and session information
  • Press A to audition tracks or folders quickly
  • Press TAB to jump to specific times or take numbers
