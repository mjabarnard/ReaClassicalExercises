Introduction

This exercise introduces you to setting up a ReaClassical project with vertical workflow, configuring input routing, track naming, and the take recording system. It's is designed to be completed in the Lyons Studio using the installed Decca tree in SJLCH for audio input (we don't need to adjust the tree for this exercise), but feel free to use different mic inputs if available.

Mac Function Key Note: On the studio Mac, you'll need to press fn + the function key (e.g., fn+F8) unless the keyboard settings have been changed to use function keys without the fn modifier.
Understanding Vertical Workflow: The vertical workflow arranges takes in vertical folders - source folders for recording takes and a destination folder for your final edit. This arrangement makes it easy to compare different performances and compile parts of each take into a final version without excessive horizontal scrolling. This approach is common in classical music production and has been implemented in ReaClassical for REAPER.

Part 1: Project Setup with Vertical Workflow

Project Structure: When setting up a project in ReaClassical, you're creating a structure designed for classical music recording and editing. The destination folder (top folder) will eventually contain your final edited version. The source folders below it will contain your individual takes, providing the raw material for your edits. All tracks are linked to a set of mixer tracks that control their settings globally. Always make changes to the mixer tracks or destination folder first, then sync with fn+F8 to propagate these changes to all source folders.
  1. Start ReaClassical and create a vertical workflow:
    1. Start ReaClassical and immediately save your project (Command+S) with a descriptive name like "DeccaTree_Session1"
    2. Press fn+F8 to set up a vertical workflow
      • When prompted, enter "3" for the number of tracks - this will create three microphone channels in each folder
    3. Name the tracks "Left", "Centre", and "Right" for the Decca Tree (this naming convention helps with organisation)
    4. Select "Yes" to auto-set recording inputs based on track names
      • Check it assigns inputs 1-3 for the three tracks when prompted (these correspond to the physical Decca Tree microphones)
    5. When asked about special tracks, select "Yes", find "Submix", enter "1" for the number. We'll name it and set it up later - we only need to do this for the Decca to collect the 3 inputs into a stereo buss
    6. Take a moment to look at how ReaClassical has structured your project
  2. Examine your project structure in detail:
    1. Notice the destination folder at the top - this is where your final edited version will eventually be built
    2. Below are six source folders, all with the same track structure - these will hold your individual takes
    3. Each folder contains the same channels (Left, Centre, Right) - this mirroring is key to the vertical workflow
      • Additional source folders will be automatically created as needed when recording multiple takes
    4. Look at the mixer panel to see the mixer tracks (prefixed with "M:") which control all folder tracks
      • The # track appears in the mixer - this is your submix that will combine all Decca Tree elements
    5. Try clicking on different folders to see how navigation works in the vertical arrangement
  3. Verify and understand input routing:
    1. Right-click on the meters of each track in the destination folder to open the input selection menu
      • Inspect the input designation - you should see each track properly assigned to inputs 1-3
      • Notice how all three tracks are correctly routed to inputs 1-3 without manual configuration
      • This happens automatically due to track naming and the auto-routing feature
    2. If you need to adjust any inputs, you can use the routing matrix as detailed in the next section
    3. Important: Always sync fn+F8 after changing routing to ensure consistent settings across all folders
  4. Exploring the routing matrix for input control:
    1. Press Option+R to open the routing matrix
      • Notice how you can see all tracks and their connections in a single view
    2. The routing matrix provides an alternative method for selecting and managing microphone inputs
      • This is particularly useful when you need to access inputs from the Dante preamps or other input sources
    3. Notice the 'Source' and 'Destination' rows and columns, you can see and modify microphone input assignments by clicking on different channels
      • You can create mono inputs or stereo pairs
    4. Make any needed input changes in the destination group or mixer tracks
    5. After making changes, press fn+F8 to propagate these changes to all source folders
    6. Important: Always sync fn+F8 after changing routing to ensure consistent settings across all folders
    7. Close the routing matrix window when finished by clicking the X or pressing Escape
  5. Add a new track and understand track management:
    1. Press Option+T to add a track to all folders simultaneously
    2. When prompted, enter the track name "Audience" - this could represent an audience microphone in a live recording
      • Notice how the track appears in every folder and in the mixer
    3. Practice selecting a stereo input for this track using either the right-click approach or the Routing Matrix (using the Stereo/MC part of the input list)
    4. Now try adding another track called "Soloist" using the same method. Select a mono input for this
    5. Important: Always sync fn+F8 after changing routing to ensure consistent settings across all folders
    6. Observe how both new tracks maintain the same position in every folder - maintaining consistency across your project
    Tip: Using the global approach to add tracks ensures consistency across your entire project, saving time and reducing the likelihood of errors that could occur when configuring tracks individually.
  6. Rename tracks and understand mixer synchronisation:
    1. In the mixer panel, rename "M:Left" to "M:Decca Left" by double-clicking on the name
    2. Similarly rename "M:Centre" to "M:Decca Centre" and "M:Right" to "M:Decca Right"
    3. Also rename "M:Audience" to "M:Hall Ambient" and "M:Soloist" to "M:Solo Spot"
      • Notice how the names in the mixer panel haven't yet changed in the main project view
    4. Press fn+F8 to sync these new names across all folders
    5. Observe how all tracks in all folders update to reflect your new naming scheme
    6. Important: Always make changes to mixer tracks (M: prefixed) or destination folder tracks first, then sync with fn+F8 to propagate these changes
  7. Setting up the Decca Tree Submix with routing:
    1. Look at the "#" submix track that was created during setup - this will become your main Decca Tree mix
    2. First, rename it to '#Decca Mix' by double-clicking the name. We need to modify our track names to properly route to the submix:
    3. In the mixer panel, rename "M:Decca Left" to "M:Decca Left-", "M:Decca Centre" to "M:Decca Centre-", and "M:Decca Right" to "M:Decca Right-"
      • The hyphen at the end of each name is crucial - it tells ReaClassical that these tracks shouldn't route directly to the master bus
    4. Press fn+F8 to sync these name changes to all folders
    5. Now in the mixer panel, click and drag from the routing button (at the bottom of the track panel) of "M:Decca Left-" to "#Decca Mix"
      • A routing window appears - simply click OK to accept the default routing
      • Similarly route "M:Decca Centre-" and "M:Decca Right-" to "#Decca Mix"
    6. This creates a network where all three Decca Tree microphones feed into a single submix
    7. Now let's configure the spatial positioning of these microphones in the stereo field:
      • In the mixer panel, select "M:Decca Left-" and adjust its pan control fully left
      • Select "M:Decca Right-" and adjust its pan control fully right
      • Leave "M:Decca Centre-" centred in the stereo image
      • This creates a proper stereo image that matches the physical layout of the Decca Tree
    8. Try adjusting the volume faders for each microphone to change their relative levels in the mix
    9. Press fn+F8 to sync these routing, pan, and volume settings to all folders
    10. Now the "#Decca Mix" track contains a properly configured mix of your three Decca Tree microphones
    11. When using additional input channels in future, try soloing the #Decca Mix track to hear how it sounds with your current settings
    Tip: Submixes give you a single fader to control grouped microphones, which is convenient for both recording and editing. In larger orchestral sessions, you might create submixes for strings, woodwinds, brass, etc., allowing for balanced section control while maintaining access to individual microphones when needed. We are creating a submix in the exercise due to the Decca tree having 3 signals - we are making a 2ch buss from those 3 feeds. You don't need to make submixes for 2ch arrays or mono feeds.

Part 2: Recording and Take Management

Take Recording in ReaClassical: The Classical Take Recording system in ReaClassical handles record arming, group switching, and take organisation automatically. This allows you to focus on the music instead of the technical details. Each take is recorded into its own source folder, making organisation intuitive and editing straightforward later on. This approach is commonly used in classical recording sessions where multiple complete takes are standard practice.
  1. Set up for recording with the Take Counter:
    1. Position the edit cursor at the beginning of the timeline by pressing Home
    2. Select the Destination folder if it's not already selected
    3. Open the Take Counter window with Command+Return
      • Notice the green number showing the next take number - this tracks your recording progress
    4. Right-click on the Take Counter window to access additional options
      • Enter a session name like "DeccaSession" - this helps organise files and makes later identification easier
      • Leave the override take number setting at 0 (automatic)
      • Explore the other options in this dialog - note how you can schedule recordings or set custom take numbers
    5. Click OK to confirm your settings
  2. Record multiple takes with automatic folder management:
    1. Press fn+F9 once to arm the tracks for recording - notice how the track meters become active
      • Observe that only the tracks in the first source folder are armed
      • The Take Counter displays the current take number in green
    2. Press fn+F9 again to begin recording - the Take Counter now shows a red recording indicator
      • Record about 15-20 seconds of audio from the Decca tree in the Hall
    3. Press fn+F9 to stop recording
    4. Notice a few things happening automatically:
      • The next source folder is automatically selected and armed - ready for the next take
      • The Take Counter increments to the next take number
      • All your recorded audio is properly organised in the first source folder
    5. Record a second take using fn+F9 and observe the same automatic processes
    6. For your third take, try pressing Command+Space during recording to pause
    7. Notice that pausing keeps the same take but creates a gap in the recording
    8. Press Command+Space again to resume recording
    9. Finally press fn+F9 to stop recording
  3. Additional recording techniques:
    1. Press fn+F9 to begin recording take 4
    2. After a few seconds of recording from the Decca tree, press Option+fn+F9
      • This increments the take number without stopping recording - useful for continuous recording sessions
      • Notice how the Take Counter advances but recording continues
    3. Record a few more seconds and press fn+F9 to stop
      • Notice how ReaClassical has now created multiple takes across different source folders
      • If you look at the source folders, you'll see each contains the appropriate audio for each take
    Tip: The take increment feature (Option+fn+F9) is particularly useful during sessions with repeated attempts at difficult passages. It allows the recording to continue while still organising takes properly for later editing.
  4. Auditioning recorded material:
    1. Navigate to take 1 by pressing Return and entering "1"
    2. Hover your mouse over the "Decca Centre-" track in take 1 and press A to hear just that microphone
    3. Now hover over the "Decca Right-" track and press A to hear just that microphone
    4. Now hover over the parent folder track "Decca Left-" and press A to hear the complete mix of that take
    5. Try hovering at different positions within the waveform and pressing A - playback starts from that position
    6. Press Option+A to audition at a slower playback rate
    7. Use Space to stop playback at any time
    8. Navigate to take 2 using Return and "2", then repeat the auditioning process
      • Compare how take 1 and take 2 sound by quickly switching between them
      • Notice how auditioning automatically disables any record-armed tracks for safety
    Tip: The auditioning system allows for quick comparison between takes and microphone positions without changing track selection or using solo/mute buttons. This helps you make informed decisions by making comparisons immediate and intuitive.
  5. Ranking takes for editing efficiency:
    1. After auditioning all your takes, navigate back to take 1
    2. Press Command+= to rank it higher - notice the colour change and "+" symbol added to the item
      • Navigate to take 2 and press Command+= twice to rank it even higher (++) - see how the color intensifies
      • Navigate to take 3 and press Command+- to rank it lower - note the different colour indicating a lower-ranked take
      • Navigate back to take 1 and press Command+0 to remove any ranking - returning it to the default colour
    3. Explore the full range of rankings by applying multiple + or - rankings to different takes
      • Notice how the visual system creates an overview of take quality across your project
    Tip: Take ranking is useful during longer recording sessions and later editing stages. The visual colour coding helps identify your best and worst takes quickly. Even if you're not editing today, ranking takes as you record them preserves your initial impressions when the performances are fresh in your mind.

Part 3: Exploring ReaClassical's Additional Features

ReaClassical's Navigation and Workflow Features: ReaClassical includes tools for efficient navigation and project management. The ability to quickly locate takes by number, audition individual tracks or complete mixes with a single keypress, and switch between different view modes all contribute to a workflow optimised for classical music production. Understanding these additional features will help you work more efficiently with classical music recordings.
  1. Navigation and visibility controls:
    1. Press ` (backtick) to see the entire project horizontally - all your takes are visible at once
    2. Press Command+' to see the entire project vertically - all your folders are visible
    3. Select any parent folder track and press E to hide all child tracks (ensconce)
      • Observe how the view simplifies to show only the parent tracks - useful for getting an overview
    4. Press D to display all child tracks again - returning to detailed view
      • Try hiding children on some folders and showing them on others to create custom views
    5. Press Return and enter different take numbers to quickly jump between takes
      • Try entering take numbers with decimals (e.g., "2.5") to position the cursor partway through a take
    6. Use Q and W to navigate to the previous or next item edge - useful for moving between edits
    7. Experiment with Command+Up Arrow and Command+Down Arrow to adjust waveform zoom - see how it affects visibility without changing audio volume
  2. Working with track groups and understanding folder structures:
    1. Press \ (backslash) to duplicate a folder without items - creating a new empty source group
      • Notice how the new folder maintains the same track structure as the others
      • This is useful when you need more source groups for additional takes beyond what was automatically created
    2. Press # to open the special tracks dialog
      • Mac Workaround: If the # key doesn't work on the Mac (which is a known issue), go to the main menu and select Actions → Show Action List, then search for "special tracks" and run that action by double-clicking it
    3. Add a "RoomTone" track - useful for maintaining natural ambience between edits
    4. Add a "Reference" track - helpful for comparing your work to other recordings
      • Notice how these special tracks appear differently in the mixer and serve specific functions
    5. Import a short audio file to the reference track and audition it to understand its purpose
  3. Recording features and scheduling:
    1. Right-click on the Take Counter and explore the scheduling options
    2. Set a start time 2 minutes ahead of the current time
      • Observe how the Take Counter now displays a timer indicating when recording will begin
      • This feature is useful for timed recordings or when you need to set up before a performance begins
      • You can also set an end time or duration for automated recording sessions
    3. Cancel the scheduled recording by right-clicking the Take Counter and clearing the time fields
    4. Access the ReaClassical manual with H for detailed information on all features

Closing Remarks

Now that you've completed this exercise, you've created a functional ReaClassical project that demonstrates the fundamentals of the vertical workflow, recording management, and take organisation. Consider keeping this project as a reference template for future sessions. When you begin a new recording project, you might find it useful to:

  • Save this project as a template that you can quickly adapt for different recording scenarios
  • Refer back to this exercise when setting up more complex recording configurations
  • Use this project as a testing ground for experimenting with different routing setups or microphone arrangements

Remember that becoming proficient with any recording system requires practice. The keyboard shortcuts you've learned here will become second nature with regular use, significantly increasing your efficiency during recording and editing sessions.

Additional Tips for ReaClassical

  • The destination folder is always the first folder - it's where you'll build your final edit
  • Source folders (below the destination) store your individual takes and are automatically added as needed during recording
  • Always make changes to mixer tracks (M: prefixed) or destination folder tracks first, then use fn+F8 to sync these changes to all source folders - this workflow ensures consistent settings across your project
  • Use the Take Counter window for every recording session - it helps track take numbers for your take sheet and prevents confusion
  • The fn+F9 key is your main recording control - it handles arming, recording, and folder selection
  • The auditioning system (A key) allows quick comparison between takes and channels
  • Rank your takes during or immediately after recording while your impressions are fresh
  • Use project preferences (fn+F5) to customise ReaClassical's behaviour to your workflow
  • Save your project frequently with Command+S
  • Explore both vertical and horizontal workflows to determine which works best for different projects
  • Take time to learn the keyboard shortcuts - they can increase editing efficiency
